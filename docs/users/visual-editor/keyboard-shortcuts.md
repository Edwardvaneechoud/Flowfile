# Keyboard Shortcuts

Mac: `⌘ Cmd` / Windows/Linux: `Ctrl`

## Flow Management

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| New Flow | `⌘ + N` | `Ctrl + N` |
| Save Flow | `⌘ + S` | `Ctrl + S` |
| Run Flow | `⌘ + E` | `Ctrl + E` |
| Code Generator | `⌘ + G` | `Ctrl + G` |

## Node Operations

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Copy | `⌘ + C` | `Ctrl + C` |
| Paste | `⌘ + V` | `Ctrl + V` |
| Delete | `Delete` / `Backspace` | `Delete` |

Multi-select: `Shift` + click or drag to box-select. Copied nodes preserve positions and connections.

## Canvas

| Input | Action |
|-------|--------|
| Mouse wheel | Zoom |
| Drag canvas | Pan |
| Drag node | Move |
| `Escape` | Close menu |

## Context Menus

**Canvas** (right-click): Fit View, Zoom In/Out, Paste Node

**Node** (right-click): Run Now, Enable/Disable Cache, Copy, Delete

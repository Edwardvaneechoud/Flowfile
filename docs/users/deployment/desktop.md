# Desktop App

Run Flowfile as a native desktop application.

## Download

Download the latest release for your platform:

- **macOS**: `.dmg` installer
- **Windows**: `.exe` installer
- **Linux**: `.AppImage`

[Download from GitHub Releases â†’](https://github.com/edwardvaneechoud/Flowfile/releases)

## Installation

### macOS

1. Open the `.dmg` file
2. Drag Flowfile to Applications
3. Open from Applications folder

### Windows

1. Run the `.exe` installer
2. Follow the installation wizard
3. Launch from Start menu

### Linux

1. Make the AppImage executable:
   ```bash
   chmod +x Flowfile-*.AppImage
   ```
2. Run it:
   ```bash
   ./Flowfile-*.AppImage
   ```

## Features

- No login required (single-user mode)
- Master key auto-generated and stored securely
- All data stored locally
- Works offline

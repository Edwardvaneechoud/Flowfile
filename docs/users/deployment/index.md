# Deployment Overview

Choose how to run Flowfile based on your needs.

| Method | Best For | Multi-user |
|--------|----------|------------|
| [Desktop App](desktop.md) | Individual use, local development | No |
| [Python Package](python.md) | Scripting, CI/CD pipelines | No |
| [Docker](docker.md) | Teams, servers, production | Yes |

## Quick Comparison

**Desktop App** - Download and run. No setup required. Best for exploring Flowfile or building flows locally.

**Python Package** - `pip install flowfile`. Run flows programmatically or integrate into existing Python projects.

**Docker** - Full deployment with authentication, secrets management, and centralized storage. Best for teams and production.
